시책 엑셀 매핑
원수사 시책결과 엑셀파일을 표준화된 레이아웃으로 매핑 정의합니다
매핑 대상 선택
📋 컬럼 매핑 — 원수사 컬럼명 → 내부 표준 컬럼
원수사 컬럼명내부 표준 컬럼변환 규칙
계약번호 → CONTRACT_NO 그대로 사용
설계사사번 → EMP_ID 그대로 사용
상품구분명 → PRODUCT_TYPE 코드변환 종신/종신보험 → PRD_LIFE
납입기간 → PAY_PERIOD 코드변환 10년/10年/PAY10 → PAY_10
월납환산보험료 → MONTHLY_PREM 숫자변환 콤마 제거, 숫자로 변환
계약일자 → CONTRACT_DATE 날짜변환 YYYYMMDD → YYYY-MM-DD
지급금액 → PAY_AMT 숫자변환 콤마 제거, 숫자로 변환
🔄 이음동의어 코드변환 규칙
대상 컬럼원수사 표현값내부 코드값
PRODUCT_TYPE : 종신, 종신보험, LIFE, 終身 PRD_LIFE
PRODUCT_TYPE : CI, CI보험, CI보험(중대질병) PRD_CI
PAY_PERIOD : 10년, 10年, PAY10, 10Y PAY_10
PAY_PERIOD : 전기납, 전납, FULLPAY PAY_FULL
⚠️ 매핑 정의는 매월 변경이 있을 수 있습니다. 원수사 엑셀 업로드 전 반드시 최신 매핑으로 업데이트 확인하세요.
